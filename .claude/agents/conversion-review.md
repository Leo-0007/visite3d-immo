# Conversion Review Agent

## Landing Page Flow
- [ ] Hero communicates value prop in <5 seconds
- [ ] Demo is accessible within 1 scroll (moved above problem-solution)
- [ ] Pricing is clear and easy to compare
- [ ] CTA buttons use consistent style (--v3d-blue, rounded-lg)
- [ ] Primary CTA = "Voir les offres" or "Commander"
- [ ] Secondary CTA = "Explorer la demo 3D"
- [ ] No dead-end sections (every section has a next action)

## Pricing Page
- [ ] Price anchoring: Solo (49) -> Pack Immo (89, recommended) -> Pack Agence (399)
- [ ] "Recommande" badge on Pack Immo
- [ ] Clear feature comparison between tiers
- [ ] "Not included" items on Solo drive upsell
- [ ] Upsells strip shows add-on value
- [ ] Contact option for custom Regie needs

## Post-Payment Flow
- [ ] Confirmation page shows correct pack
- [ ] Steps timeline is clear
- [ ] Next action is obvious (film your property)
- [ ] Guide de capture is linked
- [ ] Upgrade path available (Solo -> Pack Immo)
- [ ] Contact info visible

## Trust Signals on Conversion Path
- [ ] "Sans engagement" near CTA
- [ ] "Livraison 24h" near CTA
- [ ] "Paiement securise" near CTA
- [ ] Pricing transparency (no hidden fees)

## Mobile Conversion
- [ ] CTA buttons are full-width or easily tappable on mobile
- [ ] Pricing cards stack cleanly
- [ ] Demo viewer works on touch devices
